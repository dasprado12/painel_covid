
# painel_covid

Este é um projeto de análise de dados da Sars-CoV-2.

## Docker
```
sudo docker-compose build
sudo docker-compose up
```

## Npm

### Rodar o projeto
``` 
npm install
npm run serve
```

### Criar uma build do projeto

```
npm install
npm run build
```

### Lints and fixes files
